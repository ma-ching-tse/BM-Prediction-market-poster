# 运营同学海报生成手册（8 个模板）

只要按表格填数据，然后执行对应命令即可出图。

## 1. 快速流程（通用）

1. 打开对应模板的 Lark 表，更新当期内容。
2. 保存后执行该模板命令（见下方总表）。
3. 等待命令完成。
4. 在 `output/模板名/当天日期/` 查看图片和 zip 包。

输出目录结构示例：

- `output/足球赛事/20260417/`
- `output/足球赛事/20260417/足球赛事_20260417.zip`

## 3. 每个模板怎么填（运营视角）

## 3.1 NBA 赛事（classic）

必填列：

- `date`
- `home_team`
- `away_team`

可选列（不填也可）：

- `polymarket_slug`


系统会自动做：

- 通过主客队 + 日期去抓 Polymarket 赔率并回填。
- 多语言球队名来自 `teams.csv`。

## 3.2 综合事件（comprehensive）

第 2 行主要内容：

- `main title`
- `sub title`
- `footer`
- `1 / percent_1`
- `2 / percent_2`
- `3 / percent_3`
- 可选 `4 / percent_4`

系统会自动做：

- 翻译标题、副标题、footer、卡片文案。
- 把翻译结果写回第 3 行开始（A 列是语言代码）。

## 3.3 世界杯（worldcup）

基础字段与综合事件一致。  
另外支持配置区（A20:B35）：

- `scenario_type`（支持 `worldcup_winner` / `group_winner`）
- `market_slug` 或 `market_url`
- `card_count`

系统会自动做：

- 可从 Polymarket 自动生成卡片并回填卡片文案/概率。
- 其余多语言翻译同综合事件。

## 3.4 足球赛事（football）

第 2 行主要字段：

- `title`
- `subtitle`
- `footer`
- `match1_home / match1_away / match1_date`（必填）
- `match2_home / match2_away / match2_date`（可空）
- `match3_home / match3_away / match3_date`（可空）

强烈推荐新增字段（已在表头）：

- `match1_link`
- `match2_link`
- `match3_link`

赔率抓取规则：

1. 如果填了 `matchN_link`，优先按链接/slug 抓赔率。
2. 如果链接没填或无效，回退为“主客队 + 日期”自动匹配抓赔率。
3. 抓不到时保留原赔率，不阻断出图。

说明：

- 球队多语言不走翻译 API，统一来自 `football_teams.csv`。
- 决赛只有 1 场时，只填 `match1_*` 即可，`match2_*`、`match3_*` 留空不会报错。

## 3.5 币价预测（coinprice）

按“语言行”填写（A 列是语言代码，源语言通常是 `zh-CN`）。

主要字段：

- `main title`
- `sub title`
- `footer`
- `token_1 / 1 / percent_1`
- `token_2 / 2 / percent_2`
- `token_3 / 3 / percent_3`

系统会自动做：

- 只翻译 `title/subtitle/footer`。
- `token` 和目标价格不翻译，保持原值。

## 3.6 全球预测市场（global）

读取卡片字段（最多 3 条）：

- `title`
- `stat`
- `desc`

系统会自动做：

- 翻译 `title` 和 `desc`。
- `stat` 原样保留。

## 3.7 F1 车队（f1）

第 2 行主要字段：

- `main title`
- `sub title`
- `footer`
- `team_1 / percent_1`
- `team_2 / percent_2`
- `team_3 / percent_3`

可选配置（A 列 key、B 列 value）：

- `market_slug` 或 `polymarket_url`

系统会自动做：

- 标题/副标题/footer 自动翻译并回填。
- 车队名不翻译，来自 `teams_f1.csv`。

## 3.8 F1 车手（f1driver）

第 2 行主要字段：

- `main title`
- `sub title`
- `footer`
- `driver_1 / percent_1`
- `driver_2 / percent_2`
- `driver_3 / percent_3`
- `driver_4 / percent_4`

可选配置（A 列 key、B 列 value）：

- `market_slug` 或 `polymarket_url`

系统会自动做：

- 标题/副标题/footer 自动翻译并回填。
- 车手名不翻译，来自 `drivers_f1.csv`。

## 4. 常见问题（运营版）

1. 看不到预览链接？
- 预览链接是本机地址（`localhost:3001`），需要技术同学在本机开启预览服务。
- 即使打不开预览，也不影响命令行出图。

2. 为什么图里赔率没更新？
- 先确认主客队 ID 是否正确。
- 足球模板建议优先填 `matchN_link`，准确率最高。

3. 产物在哪里找？
- 看 `output/模板名/当天日期/`。
- 里面有单张 JPG 和一个 zip 包。
