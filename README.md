# BitMart Prediction Market Poster

这是运营可直接使用的海报生成仓库（当前 8 个模板）。

## 运营入口

请先看这份手册：

- `docs/04-运营同学海报生成手册.md`

手册里包含：

- 每个模板该填什么
- 每个模板的 Lark 链接
- 一键生成命令
- 输出目录与常见问题
- 后续更新同步 SOP（不需要重新下载仓库）

## 常用命令（技术同学）

```bash
npm run generate:nba
npm run generate:comprehensive
npm run generate:worldcup
npm run generate:football
npm run generate:coinprice
npm run generate:global
npm run generate:f1
npm run generate:f1driver
```
